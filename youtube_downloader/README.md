# Youtube Downloader

Un programa simple para descargar videos y audio de YouTube.

## Requisitos

- Python 3
- FFmpeg (Ya instalado en tu sistema)

## Instalación

1. Abre una terminal en esta carpeta.
2. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

1. Ejecuta el programa:
   ```bash
   python3 app.py
   ```
2. Abre tu navegador en la dirección que aparece (probablemente http://127.0.0.1:5050).
3. Pega el link de YouTube y descarga.

Los archivos se guardarán en la carpeta `downloads` dentro de este directorio.
